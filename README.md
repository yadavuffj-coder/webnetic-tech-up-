# WEBNETIC TECH UP - Online Learning Platform

A modern, professional, mobile-friendly online learning platform for students and teachers.

## Features

- **Live Classes**: Real-time learning with Jitsi Meet integration
- **Recordings**: Automatic class recordings for on-demand learning
- **Course Management**: Organize content into Courses → Subjects → Chapters
- **Student Portal**: Access courses, join live classes, watch recordings, download notes
- **Teacher Portal**: Create content, manage classes, upload recordings and notes
- **Admin Dashboard**: Complete platform management and analytics
- **Mobile Optimized**: Fully responsive for Android, iPhone, Tablet, and Desktop

## Tech Stack

### Frontend
- React.js / Next.js
- Tailwind CSS
- Dark/Light Mode

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- MongoDB

### Additional Services
- Jitsi Meet (Live Classes)
- Cloudinary / AWS S3 (File Storage)
